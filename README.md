# BMI Calculator

This is a **BMI (Body Mass Index) Calculator Application** developed for academic/project purpose.

The application takes **height and weight** as inputs and calculates the BMI value using the formula:
BMI = Weight (kg) / Height (mÂ²)


It also displays the corresponding BMI category:
- Underweight
- Normal
- Overweight
- Obese
